5.33 acc_get_current_cuda_context – Get CUDA context handle.

Description

This function returns the CUDA context handle. This handle is the same as used by the CUDA Runtime or Driver API’s.

C/C++:
Prototype:acc_get_current_cuda_context(void);
Reference:

OpenACC specification v2.0, section A.2.1.2.
